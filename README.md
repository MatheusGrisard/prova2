# TechZone

Projeto front-end de uma loja virtual fictícia especializada em periféricos gamer e equipamentos para setup. A aplicação foi desenvolvida com HTML, CSS e JavaScript, com páginas de navegação, listagem de produtos, carrinho, login, cadastro e formulário de contato.

## Objetivo do projeto

Criar uma interface web para simular a experiência de compra em um e-commerce gamer, aplicando conceitos de estruturação HTML, estilização com CSS e interatividade com JavaScript.


## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Estrutura de pastas

## Páginas do projeto

- `index.html`: página inicial da TechZone.
- `sobre.html`: informações institucionais da loja.
- `produtos.html`: catálogo de produtos com busca e filtros.
- `carrinho.html`: página do carrinho de compras.
- `contato.html`: formulário e canais de atendimento.
- `login.html`: tela de acesso do usuário.
- `register.html`: tela de cadastro de usuário.

## Funcionalidades

- Página inicial com banner, categorias e produtos em destaque.
- Página "Sobre" com apresentação da empresa, missão, visão e valores.
- Página de produtos com busca, filtros por categoria e cards de itens.
- Carrinho de compras com cálculo de valores.
- Páginas de login e cadastro de usuário.
- Formulário de contato com informações de atendimento.
- Navegação entre páginas por menu superior.
- Armazenamento de informações no `localStorage`.
